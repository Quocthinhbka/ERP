# Report — Admin: mã tài khoản Admin, xoá SĐT/Email

- Thời điểm: 2026-07-23 11:32:00 (local)

## Yêu cầu thay đổi

Đặt **mã tài khoản** của Super Admin hiện tại thành **Admin**, và **xoá SĐT + Email** trên tài khoản đó.

## Việc đã thực hiện

1. **Cập nhật DB** (user `is_super_admin = true`):
   - `account_code` → `Admin`
   - `email` → `NULL`
   - `phone` → `NULL`

2. **`auth.service.ts`**: đăng nhập bằng mã tài khoản dùng `findFirst` **không phân biệt hoa thường** (thay `toUpperCase()` + `findUnique`) để mã `Admin` hoạt động ổn định.

3. Rebuild API + restart dev servers.

## Kết quả thực hiện

- **Thành công** — đăng nhập bằng `Admin` + mật khẩu hiện tại (`.env` `TEST_ADMIN_PASSWORD`) trả `email: null`, `accountCode: "Admin"`.
- **Lưu ý**: không còn đăng nhập bằng email/SĐT cũ trên admin này. E2E/local dùng `TEST_ADMIN_EMAIL` sẽ lỗi cho đến khi reset DB (bootstrap tạo admin mới) hoặc đổi helper login sang mã `Admin`.
- File `.env` (`TEST_ADMIN_EMAIL`/`TEST_ADMIN_PHONE`) **chưa xoá** — chỉ dùng khi bootstrap DB trống; admin hiện tại trong DB đã không còn email/phone.
