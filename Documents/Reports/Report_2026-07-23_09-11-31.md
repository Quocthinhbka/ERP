# Up/Down sắp xếp trường trên tab hồ sơ

**Thời gian:** 2026-07-23 09:11:31

## Tóm tắt

Trên **Thiết lập hồ sơ nhân sự**, mỗi trường đã gắn tab có nút **Lên / Xuống** ở đầu dòng để đổi thứ tự hiển thị. Gọi `PUT /employee-profile-settings/tabs/:tabId/fields/reorder`.

## File đổi

- `apps/web/src/pages/setup/EmployeeProfileFieldSettingsPage.tsx`

## Hành vi

- Nút Up disable ở dòng đầu; Down disable ở dòng cuối.
- Chỉ hiện khi có quyền `SETUP_MANAGE`.
- Thứ tự lưu trên link tab–field (`sortOrder`), form/chi tiết hồ sơ dùng layout đã sắp.
