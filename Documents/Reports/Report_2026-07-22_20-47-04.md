# Report: Sửa import/export hồ sơ nhân viên (JSON/Excel)

**Thời gian:** 2026-07-22 20:47:04

## Yêu cầu thay đổi

- Sửa lỗi Prisma `Invalid value for argument gender. Expected EmployeeGender` khi import (ô trống thành `""`).
- Cho phép chọn định dạng **JSON** hoặc **Excel** khi xuất/nhập thông tin nhân viên (giống tổ chức).

## Việc đã thực hiện

### Package `@erp/employee-io`
- `validate.ts`: normalize `gender` / `educationLevel` / `religion` / ngày tháng — chuỗi rỗng → `null`.
- `apply.ts`: `emptyToNull` trước khi ghi Prisma.
- `excel.ts`: đọc ô enum trống thành `null`.
- Thêm `json.ts` (serialize / parse / detect format).
- `jobs.ts`: export/template/diff theo `excel | json`.
- Unit test: normalize enum rỗng, JSON round-trip, Excel blank gender.

### API / Queue / Worker
- `ExportEmployeeDto` + body `format`.
- Import chấp nhận `.xlsx` / `.json`.
- Download đúng MIME theo đuôi file.
- Queue/worker truyền `format` và `originalName`.

### Web
- `EmployeeIoActions`: modal chọn Excel/JSON khi xuất / tải mẫu / nhập.

### Test
- E2E: export JSON; import JSON với `gender: null` không lỗi Prisma.
- Đã build lại package + restart `pnpm dev:up`.
