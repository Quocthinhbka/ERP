**Thời gian:** 2026-07-23 00:36:54  
**Phương án:** A (đã duyệt)

## Tóm tắt

Cải UI **Thiết lập / Nhân sự / Hồ sơ** (`EmployeeProfileFieldSettingsPage`):

1. **Thanh tab tùy chỉnh** — nút hoán đổi giữa hai tab liền kề (`PUT .../tabs/reorder`); nút **Thêm Tab** tạo tab tên mặc định「Đổi tên Tab」, focus + chọn toàn bộ chữ để đổi tên inline (`POST` + `PATCH`), không dùng modal.
2. **Hai vùng trường theo tab đang chọn**
   - Trên: trường đã gắn — **Bỏ khỏi tab** → detach API, hiện lại vùng dưới.
   - Dưới: thư viện trường chưa gắn tab này — **Chọn** → attach API, lên vùng trên.
3. Giữ sửa/xóa trường, checkbox bắt buộc/hiển thị; double-click tab để đổi tên; xóa tab non-system.

## File đổi

- `apps/web/src/pages/setup/EmployeeProfileFieldSettingsPage.tsx` — UI mới

## API dùng sẵn

- `POST /employee-profile-settings/tabs`
- `PATCH /employee-profile-settings/tabs/:id`
- `PUT /employee-profile-settings/tabs/reorder`
- `POST /employee-profile-settings/tabs/:tabId/fields`
- `DELETE /employee-profile-settings/tabs/:tabId/fields/:fieldDefId`
